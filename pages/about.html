About

Working on harness engineering from San Francisco and Palo Alto.

My current interests are agent memory, harness engineering, and knowledge graphs.

I am focused on building the infrastructure around agent systems: the memory layers that determine what an agent can retain, the harnesses that let you evaluate behavior under real constraints, and the graph structures that make context more durable and inspectable.

This site is also a working system. It is intentionally static-first, with authored HTML, isolated client bundles, and a custom build pipeline that keeps the runtime surface area small.
